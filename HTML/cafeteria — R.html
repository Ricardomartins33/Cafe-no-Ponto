    Para mudar a cor do botão quando ele é clicado, você pode usar JavaScript junto com o CSS. Vamos adicionar um evento de clique ao botão e, sempre que ele for clicado, mudaremos a cor de fundo. Aqui está um exemplo de como fazer isso:

    HTML

    <button class="button-main">Clique Aqui</button>

    CSS

    .button-main {
    background: #ADFF00;
    border-radius: 30px;
    width: 239px;
    height: 64px;
    border: none;
    cursor: pointer;
    filter: drop-shadow(0px 4px 4px rgba(0, 0, 0, 0.25));
    font-weight: 700;
    font-size: 28px;
    line-height: 34px;
    color: #000000;
    transition: background 0.3s; /* Adiciona uma transição suave /
}

.button-main.clicked {
    background: #FF4500; / A nova cor quando clicado */
}

    JavaScript

    document.querySelector('.button-main').addEventListener('click', function() {
    this.classList.toggle('clicked'); // Adiciona ou remove a classe 'clicked' ao clicar
});

    Explicação

    1. HTML: Crie um botão com a classe button-main.
    2. CSS: Defina o estilo normal do botão e um estilo alternativo com a classe clicked. A propriedade transition permite que a mudança de cor aconteça de forma suave.
    3. JavaScript: Adicione um evento de clique que alterna (toggle) a classe clicked no botão. Isso mudará a cor do botão quando ele for clicado.
    Com isso, toda vez que o botão for clicado, a cor de fundo mudará para a cor definida na classe clicked. Se você clicar novamente, a cor voltará à original.
